Flutix is ​​a mobile application (iOS & Android) that helps people to buy tickets to watch online through their mobile phones.

Flutix also uses the data collection method to display movies that match the user's profile.

Features
  • User Management (login/sign up)
  • Browse Movies Category
  • Choose Movies from theMovieDB Public API
  • Select Bioskop
  • Select Date & Time
  • E-wallet for ticket transaction
  • Ticket Data Collection
